# Shadowsocks-Qt5-Installation-on-Debian
Installation Script For shadowsocks-Qt5 on Ubuntu/Debian

## Usage
``` bash
git clone git@github.com:FLHonker/fl-bash.git
cd ssr-qt5-install
sudo chmod +x ssr-qt-install.sh
./ssr-qt-install.sh
```
